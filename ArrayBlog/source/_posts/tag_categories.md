Hexoで記事にタグ付けしたり、カテゴリ分けをする方法

## 設定方法
記事である.mdのtitleの下に追記する。

### カテゴリを設定

``` 
categories:
    - [ブログ作成]
    - [Hexo]
```

### タグを設定

``` 
tags:
    - [Hexo]
```
